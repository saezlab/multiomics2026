# Scripts for preparing initial data files for the project

These scripts are not part of the project itself, but have been used to extract
data from the paper's "Extended View" (supplementary) tables, and from the
`Rdata` files in the [repository accompanying the paper
](https://github.com/saezlab/kidneyfibrosis_multiomicsmodel_paper).

If you use them, set manually the directory paths in the beginning.
